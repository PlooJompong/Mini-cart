An interactive block with the Interactivity API.

== Description ==

An interactive mini cart block powered by the WordPress Interactivity API. This block fetches and displays cart data dynamically from the WooCommerce Store API, providing a seamless and modern shopping experience. Lightweight, customizable, and easy to integrate into any WooCommerce store.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/mini-cart` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
